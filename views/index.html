Timestamp Microservice version 2.0

Pass a valid date string to GET /api/timestamp/:date_string?

Valid dates are able to be parsed by the javascript function new Date(date_string).

Some valid dates include:

  • An integer representing the number of milliseconds since January 1, 1970, ex 100000000
  • A natural date string, ex. November 01, 2010

Example input:

  • https://timestamp-generator.glitch.me/api/timestamp/
  • https://timestamp-generator.glitch.me/api/timestamp/1288569600
  • https://timestamp-generator.glitch.me/api/timestamp/July 13, 2018 15:32:00 GMT

Example output:

  • { "unix": 1531495920000, "utc": "Fri, 13 Jul 2018 15:32:00 GMT" }

Timestamp Microservice version 1.0

Pass in either a unix timestamp or a natural language date
The API will return both the unix timestamp and natural language date as a JSON object.

Example Input:

https://timestamp-generator.glitch.me/1288569600
https://timestamp-generator.glitch.me/November 01, 2010
https://timestamp-generator.glitch.me/Janeiro 15, 2018

Example Output:

{"unix":"1288569600","natural":"November 1, 2010"}
