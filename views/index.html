Timestamp Microservice

Pass in either a unix timestamp or a natural language date
The API will return both the unix timestamp and natural language date as a JSON object.

Example Input:

https://timestamp-generator.glitch.me/1288569600
https://timestamp-generator.glitch.me/November 01, 2010
https://timestamp-generator.glitch.me/Janeiro 15, 2018

Example Output:

{"unix":"1288569600","natural":"November 1, 2010"}
